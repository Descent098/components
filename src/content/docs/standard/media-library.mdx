## Usage

There are no dependencies to use this component

### Props

Below are the props you can pass to the component to control it's behaviour.

<Aside type="caution">
When using web components attributes passed in from HTML can **only** be strings, so for non-string types you need to pass values in a `JSON.parse()` compatible format. For example on an element `my-element` imagine the attributes:

- numbers which is type `Number[]`
- data which is of type `Object[]` where object has 2 `Number`s called weight and age and a `string` called name

You would do:

```html
<my-element 
    numbers="[1,2,3,4]" 
    data='[{"weight":205, "age":30, "name":"james"}]'
    ></my-element>
```

If you run into issues when using a component, try chucking your string into a `JSON.parse()` and if it errors, the problem is how you're passing your data.
</Aside>

#### Required

| Name | Argument Type | Description |
|------|---------------|-------------|
| data | `Object[]` | The data to use for the media. Should be a list of objects representing the items (i.e. `[{title:"my Movie", description:"description here", image:"url here"}, {title:"my Movie", description:"description here", image:"url here", link:"some url here"}]`) | 

#### Optional

| Name | Argument Type | Default Value | Description |
|------|---------------|---------------|-------------|
| aspectratio | `string` | `"8/12"` |  The aspect ratio of each item |
| initiallygreyscale | `boolean` | `true` | Whether to have the greyscale effect before hover |
