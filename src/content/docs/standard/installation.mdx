## What Are Web Components?

The web components standard[^1] [^2] is a browser standard that allows people to define their own custom components they can use. They are defined in javascript, but can be used like regular HTML elements. They tend to look something like this:

```js
class MyElement extends HTMLElement{
    constructor() {
        super();
    }
    ... // More code here
}

customElements.define("my-element", MyElement)
```

After running the above code you would be able to use the element with:

```html
<my-element></my-element>
```
