This card is designed to nicely fade between the image-based view and solid colour view to reveal the description. Great for situations where you only want to display details for specific items users would be interested in

## Usage

In order to use this component you will need

<RequiredTech technologies={["react", "TailwindCSS"]} noInstall={true}/>

A guide for setting up the stack can be found [here](./../installation)

### Props

Below are the props you can pass to the component to control it's behaviour.

#### Required

| Name | Argument Type | Description |
|------|---------------|-------------|
| headline | `string` | The card Title |
| smallText | `string` | The text that shows up on initial load |
| description | `string` | The text that appears on click |
| image | `string` | The image to use for the background |
| imageDescription | `string` | The alt text for the image |
