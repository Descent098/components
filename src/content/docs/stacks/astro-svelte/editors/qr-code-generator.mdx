## Usage

In order to use this component you will need

<RequiredTech technologies={["astro","svelte"]} noInstall={true} noTitle={true}/>

A guide for setting up the stack can be found [here](./../../installation)

Then install the required packages:

```bash
npm install qrcode culori
```

### Props

Below are the props you can pass to the component to control it's behaviour.

#### Required

There are no required props for this component

#### Optional 

| Name | Argument Type | Default Value | Description |
|------|---------------|---------------|-------------|
| link | `string` | `"https://"` | The default URL you want to link to with the QR code | 
| qrColor | `string` | `"#141414"` | The default color to use for the QR code |
| backgroundColor | `string` | `"#f0f0f0"` | The default color to use for the background of the QR Code |
| scale | `string\|Number` | `4` | The default number to scale up (or down) the image by |
| error | ` "L" \| "M" \| "H" \| "Q" ` | `"M"` | The default error type to use |
| version | `Number` (0-40) | `undefined` | The default QR version to use (`undefined` is auto) |
| maskPattern | `Number` (0-7) | `undefined` | The default mask pattern to use (`undefined` is auto) |
